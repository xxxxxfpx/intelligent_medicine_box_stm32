# DS1302 RTC驱动调试记录

## 硬件连接

| STM32引脚 | DS1302引脚 | 功能 |
|-----------|-----------|------|
| PB12 | RST/CE | 片选 |
| PB13 | DAT/I/O | 数据 |
| PB14 | CLK/SCLK | 时钟 |

## 调试过程

### 问题1: 显示 00 00 00

- **原因**: 引脚配置错误，原来使用PA4/PA5/PA6
- **解决**: 更换为PB12/PB13/PB14

### 问题2: 显示 FF FF FF

- **原因**: 通信成功但写入失败
- **解决**: 添加延时，确保时序正确

## 关键发现

1. **DS1302秒寄存器Bit7是时钟暂停位(CH)**
   - CH=1: 时钟停止
   - CH=0: 时钟运行

2. **写入前必须清除写保护位(WP)**
   - 控制寄存器地址: 0x8E(写) / 0x8F(读)
   - WP=0: 允许写入
   - WP=1: 禁止写入

3. **时序需要足够延时**
   - CE to CLK Setup: 4µs (at 2.0V)
   - CLK Low/High Time: 1000ns (at 2.0V)

## DS1302寄存器地址表

| 寄存器 | 写地址 | 读地址 | Bit7 | 范围 |
|--------|--------|--------|------|------|
| 秒 | 0x80 | 0x81 | CH (时钟暂停) | 00-59 |
| 分 | 0x82 | 0x83 | - | 00-59 |
| 时 | 0x84 | 0x85 | 12/24 | 0-23 |
| 日 | 0x86 | 0x87 | - | 1-31 |
| 月 | 0x88 | 0x89 | - | 1-12 |
| 星期 | 0x8A | 0x8B | - | 1-7 |
| 年 | 0x8C | 0x8D | - | 00-99 |
| 控制 | 0x8E | 0x8F | WP (写保护) | - |
