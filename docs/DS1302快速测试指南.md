## 🔌 接线确认

### OLED接线
```
OLED模块    →  STM32F103C8T6
────────────────────────────
VCC         →  3.3V 或 5V
GND         →  GND
SCL         →  PB8
SDA         →  PB9
```

### DS1302接线
```
DS1302模块  →  STM32F103C8T6
────────────────────────────
VCC         →  3.3V 或 5V
GND         →  GND
CLK (SCLK)  →  PA5
DAT (I/O)   →  PA6
RST (CE)    →  PA4
```

**重要**：请确保DS1302模块的电池座中插入了CR2032纽扣电池！

## 🔍 测试结果判断

### ✅ 成功标志
- OLED显示 "DS1302 OK!"
- 时间每秒递增（12:00:30 → 12:00:31 → 12:00:32...）
- 日期显示正确（2026-04-09）

### ❌ 失败表现

#### 现象1：OLED不显示
**检查**：
- OLED接线（PB8, PB9）
- OLED供电

#### 现象2：一直显示 "Init DS1302..."
**检查**：
- DS1302接线（PA4, PA5, PA6）
- DS1302电池是否插入
- 电池电压是否正常

#### 现象3：时间显示不正确
**原因**：
- 可能是首次上电，DS1302内部数据随机
- 测试程序会自动设置时间为：2026-04-09 12:00:00
- 如果时间不递增，检查接线

## 💡 调试技巧

### 1. 先测试OLED
如果OLED不显示，先确保OLED正常工作，再测试DS1302。

### 2. 检查接线顺序
确保杜邦线连接牢固，没有错位。

### 3. 测量电压
- DS1302 VCC应有3.3V或5V
- 电池座应有3V左右电压

### 4. 逐步调试
如果有问题，可以先简化代码，只初始化DS1302和显示固定字符串。
