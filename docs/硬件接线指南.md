# 称重模块接线指南

## ⚠️ 重要说明

本项目提供了**两种**称重实现方案：

### 方案一：HX711专用模块方案（推荐）

使用HX711专用24位高精度A/D转换模块。

### 方案二：STM32内部ADC方案（项目中实际使用）

直接使用STM32F103C8T6的内部ADC进行称重传感器信号采集。

---

# 一、HX711专用模块方案

## 模块简介

HX711是一款高精度24位A/D转换器，专门用于称重传感器信号采集。

**规格参数**：
- 工作电压：2.6-5.5V
- A/D转换精度：24位
- 通信接口：串行时钟（PD_SCK）和数据输出（DOUT）
- 推荐时钟频率：≤1MHz

---

## 引脚定义

### HX711模块引脚（5引脚）

| 引脚名称 | 功能说明 | 颜色标识（常见） |
|----------|----------|-----------------|
| VCC | 电源正极（2.6-5.5V） | 红色 |
| GND | 电源负极 | 黑色 |
| DT | 数据输出引脚 | 绿色/白色 |
| SCK | 串行时钟引脚 | 黄色/蓝色 |
| E+ | 传感器正极（供桥电压+） | - |
| E- | 传感器负极（供桥电压-） | - |
| A+ | 放大器正输入 | - |
| A- | 放大器负输入 | - |

## STM32F103C8T6连接方案

### 推荐方案（使用GPIO软件模拟）

#### 引脚连接表

| HX711引脚 | STM32F103C8T6引脚 | 说明 |
|-----------|-------------------|------|
| VCC | 3.3V 或 5V | 推荐5V供电，稳定性更好 |
| GND | GND | 共地 |
| DT (DATA) | PC0 (或任意GPIO) | 数据输出引脚 |
| SCK (CLK) | PC1 (或任意GPIO) | 时钟控制引脚 |

#### STM32引脚配置建议

```
推荐使用GPIOC端口：
- DT  → PC0 (GPIOC, Pin 0)
- SCK → PC1 (GPIOC, Pin 1)

优点：连续引脚，便于代码管理
```

## 软件配置

### GPIO配置（HAL库）

```c
// HX711引脚定义
#define HX711_DT_PIN   GPIO_PIN_0   // PC0 - 数据引脚
#define HX711_SCK_PIN  GPIO_PIN_1   // PC1 - 时钟引脚
#define HX711_PORT     GPIOC

// GPIO初始化配置
void HX711_GPIO_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStruct = {0};
    
    // 使能GPIOC时钟
    __HAL_RCC_GPIOC_CLK_ENABLE();
    
    // 配置SCK为推挽输出
    GPIO_InitStruct.Pin = HX711_SCK_PIN;
    GPIO_InitStruct.Mode = GPIO_MODE_OUTPUT_PP;
    GPIO_InitStruct.Pull = GPIO_NOPULL;
    GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_LOW;
    HAL_GPIO_Init(HX711_PORT, &GPIO_InitStruct);
    
    // 配置DT为浮空输入
    GPIO_InitStruct.Pin = HX711_DT_PIN;
    GPIO_InitStruct.Mode = GPIO_MODE_INPUT;
    GPIO_InitStruct.Pull = GPIO_NOPULL;
    HAL_GPIO_Init(HX711_PORT, &GPIO_InitStruct);
}
```

## 通信协议

### 时序要求

1. **数据准备好信号**：当DT引脚为低电平时，表示数据已准备好
2. **时钟脉冲**：SCK引脚产生脉冲读取数据
3. **数据格式**：24位有符号整数（补码形式）
4. **采样速率**：
   - A通道：10Hz（引脚25连接GND）或80Hz（引脚25连接VCC）
   - B通道：40Hz（固定）

### 数据读取流程

```
1. 等待DT变为低电平（数据准备好）
2. 连续输出24个时钟脉冲
3. 在每个时钟的下降沿读取DT引脚状态
4. 组合成24位数据
5. 第25个脉冲用于选择下一个通道
6. 将SCK保持高电平至少60μs选择新通道
```

---

## 注意事项

### 电源要求
- **推荐电压**：5V供电
- **原因**：HX711内部使用5V作为参考电压，5V供电时精度最高
- **注意**：STM32的GPIO电平为3.3V，与HX711兼容

### 布线建议
1. **电源线**：使用短而粗的电源线，减少压降
2. **信号线**：DT和SCK信号线应尽量短，远离强干扰源
3. **屏蔽线**：长距离传输时考虑使用屏蔽线
4. **去耦电容**：在HX711模块VCC和GND之间并联100μF和0.1μF电容

### 接地处理
- STM32、HX711模块、称重传感器必须共地
- 使用单点接地，避免地环路干扰

### 称重传感器安装
1. 确保传感器受力均匀
2. 避免侧向力和剪切力
3. 传感器与安装面接触良好
4. 预加载一定的压力以消除机械间隙

## 测试程序

### 基础读取测试

```c
// 读取HX711数据（24位）
long HX711_Read(void)
{
    unsigned long count = 0;
    unsigned char i = 0;
    
    // 等待DT变为低电平
    while(HAL_GPIO_ReadPin(HX711_PORT, HX711_DT_PIN));
    
    delay_us(1);
    
    // 读取24位数据
    for(i = 0; i < 24; i++)
    {
        HAL_GPIO_WritePin(HX711_PORT, HX711_SCK_PIN, GPIO_PIN_SET);
        count = count << 1;
        delay_us(1);
        
        if(HAL_GPIO_ReadPin(HX711_PORT, HX711_DT_PIN))
        {
            count++;
        }
        
        HAL_GPIO_WritePin(HX711_PORT, HX711_SCK_PIN, GPIO_PIN_RESET);
        delay_us(1);
    }
    
    // 第25个脉冲，设置下一个通道
    HAL_GPIO_WritePin(HX711_PORT, HX711_SCK_PIN, GPIO_PIN_SET);
    delay_us(1);
    HAL_GPIO_WritePin(HX711_PORT, HX711_SCK_PIN, GPIO_PIN_RESET);
    delay_us(1);
    
    // 转换为有符号数
    count = count ^ 0x800000;
    
    return (long)count;
}
```

## 常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|---------|
| 数据跳动大 | 电源不稳定 | 使用稳定的5V电源 |
| 读数一直为0 | 接线错误 | 检查DT和SCK接线 |
| 读数偏大 | 传感器过载 | 减小负载 |
| 无数据输出 | 传感器未连接 | 检查E+/E-/A+/A-连接 |
| 数据漂移 | 环境温度变化 | 预热30分钟后使用 |
| 读数不准确 | 未校准 | 进行校准操作 |

---

## 校准步骤

1. 清空称重平台，确保无任何物体
2. 记录零点读数（连续采样10次取平均）
3. 放置已知重量的标准砝码
4. 记录砝码读数
5. 计算校准系数：`系数 = 标准重量 / (砝码读数 - 零点读数)`
6. 存储校准系数到Flash或EEPROM
