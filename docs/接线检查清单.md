## 测试前检查表

请在测试前逐项确认：

1. [ ] 所有杜邦线连接牢固
2. [ ] 没有相邻引脚短路
3. [ ] VCC和GND接线正确（避免反接）
4. [ ] OLED模块已正确供电
5. [ ] DS1302模块已正确供电
6. [ ] DS1302纽扣电池已插入
7. [ ] STM32开发板已通过ST-Link连接电脑

## 常见接线错误

### ❌ 错误1：VCC和GND反接
**后果**：模块烧毁
**检查**：红线接VCC，黑线接GND

### ❌ 错误2：相邻引脚短路
**后果**：通讯失败
**检查**：用万用表测量相邻引脚是否导通

### ❌ 错误3：DS1302电池未插入
**后果**：断电后时间归零
**检查**：打开DS1302模块电池座，确认电池已插入

### ❌ 错误4：杜邦线松动
**后果**：通讯不稳定
**检查**：轻拉杜邦线，确认连接牢固
