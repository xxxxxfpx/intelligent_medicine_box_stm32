## 各模块说明

### 1. 启动文件 (startup_stm32f10x_md.s)

**位置**: `Start/startup_stm32f10x_md.s`

**功能**:
- 定义中断向量表
- 初始化堆栈指针 (SP)
- 系统复位后首先执行的代码

**启动流程**:
```
上电/复位
    │
    ▼
┌─────────────┐
│  复位中断    │
│ Reset_Handler│
└──────┬──────┘
       │
       ├─► 调用 SystemInit()  // 初始化系统时钟
       │
       ├─► 调用 main()        // 进入主程序
       │
       ▼
   用户程序
```

### 2. 系统初始化 (system_stm32f10x.c/.h)

**位置**: `Start/system_stm32f10x.c`

**功能**:
- 定义 `SystemInit()` 函数
- 配置系统时钟（SYSCLK）、AHB总线时钟（HCLK）、APB1/APB2总线时钟
- 设置Flash等待周期
- 使能外部晶振（HSE）并配置PLL

**典型时钟配置**:
```
外部晶振 HSE = 8MHz
        │
        ▼
    PLL倍频 x9
        │
        ▼
    SYSCLK = 72MHz (系统时钟)
        │
    ┌───┴───┐
    ▼       ▼
  AHB分频   APB2分频
    │       │
    ▼       ▼
  HCLK    PCLK2
  72MHz   72MHz
```

### 3. 主程序 (main.c)

**位置**: `User/main.c`

**功能**:
- 用户应用程序入口
- 初始化外设（GPIO、USART、TIM等）
- 实现业务逻辑

**典型结构**:
```c
int main(void) {
    // 1. 变量声明（必须放在函数开头）
    GPIO_InitTypeDef GPIO_InitStructure;
    
    // 2. 使能外设时钟
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOC, ENABLE);
    
    // 3. 配置外设
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_13;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_PP;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOC, &GPIO_InitStructure);
    
    // 4. 主循环
    while(1) {
        // 业务逻辑
    }
}
```

### 4. 中断处理 (stm32f10x_it.c/.h)

**位置**: `User/stm32f10x_it.c`

**功能**:
- 定义各种中断服务程序（ISR）
- 处理外部中断、定时器中断、串口中断等

**中断流程**:
```
中断触发
    │
    ▼
┌─────────────┐
│  查询向量表  │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  跳转至ISR   │
│ (stm32f10x_it.c)
└──────┬──────┘
       │
       ▼
   执行中断处理
```

### 5. 寄存器描述文件

#### stm32f10x.h
**位置**: `Start/stm32f10x.h`

**功能**:
- 定义所有外设寄存器的地址映射
- 定义寄存器位掩码和位操作宏
- 包含外设结构体定义

```c
// 示例：GPIO寄存器定义
typedef struct {
    __IO uint32_t CRL;    // 端口配置低寄存器
    __IO uint32_t CRH;    // 端口配置高寄存器
    __IO uint32_t IDR;    // 端口输入数据寄存器
    __IO uint32_t ODR;    // 端口输出数据寄存器
    __IO uint32_t BSRR;   // 端口位设置/清除寄存器
    __IO uint32_t BRR;    // 端口位清除寄存器
    __IO uint32_t LCKR;   // 端口配置锁定寄存器
} GPIO_TypeDef;
```

#### core_cm3.c/.h
**位置**: `Start/core_cm3.c/.h`

**功能**:
- Cortex-M3内核相关定义
- NVIC（嵌套向量中断控制器）操作函数
- SysTick定时器操作函数
- 内核寄存器访问函数

### 6. 标准外设库 (Library/)

**位置**: `Library/`

**功能**:
- 提供外设的库函数封装，简化开发
- 每个外设对应一对 `.c/.h` 文件

**主要库文件**:
| 文件名 | 说明 |
|--------|------|
| `misc.c/.h` | NVIC和SysTick配置 |
| `stm32f10x_gpio.c/.h` | GPIO操作 |
| `stm32f10x_rcc.c/.h` | 时钟控制 |
| `stm32f10x_usart.c/.h` | 串口通信 |
| `stm32f10x_tim.c/.h` | 定时器 |
| `stm32f10x_adc.c/.h` | ADC模数转换 |
| `stm32f10x_i2c.c/.h` | I2C通信 |
| `stm32f10x_spi.c/.h` | SPI通信 |

**使用示例**:
```c
// 使用库函数配置GPIO
GPIO_InitTypeDef GPIO_InitStruct;
GPIO_InitStruct.GPIO_Pin = GPIO_Pin_13;
GPIO_InitStruct.GPIO_Mode = GPIO_Mode_Out_PP;
GPIO_InitStruct.GPIO_Speed = GPIO_Speed_50MHz;
GPIO_Init(GPIOC, &GPIO_InitStruct);

// 设置引脚电平
GPIO_SetBits(GPIOC, GPIO_Pin_13);    // 高电平
GPIO_ResetBits(GPIOC, GPIO_Pin_13);  // 低电平
```

### 7. 库函数配置 (stm32f10x_conf.h)

**位置**: `User/stm32f10x_conf.h`

**功能**:
- 选择需要编译的库模块
- 通过注释/取消注释来启用/禁用外设

```c
// 示例：启用需要的外设
#include "stm32f10x_gpio.h"
#include "stm32f10x_rcc.h"
#include "stm32f10x_usart.h"
// #include "stm32f10x_adc.h"  // 暂时不需要，注释掉
```

## 程序执行流程

```
┌─────────────────────────────────────────────────────────────┐
│                        程序执行流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 上电/复位                                                │
│     │                                                       │
│     ▼                                                       │
│  2. 从复位向量表获取SP和Reset_Handler地址                      │
│     │                                                       │
│     ▼                                                       │
│  3. 执行 startup_stm32f10x_md.s 中的 Reset_Handler            │
│     │                                                       │
│     ├─► 3.1 初始化堆栈指针                                    │
│     │                                                       │
│     ├─► 3.2 调用 SystemInit()                                │
│     │       (system_stm32f10x.c)                             │
│     │       └─► 配置系统时钟为72MHz                           │
│     │                                                       │
│     ├─► 3.3 跳转到 main()                                    │
│     │                                                       │
│     ▼                                                       │
│  4. 执行 main.c 中的 main() 函数                              │
│     │                                                       │
│     ├─► 4.1 声明变量（C89标准要求在函数开头）                   │
│     │                                                       │
│     ├─► 4.2 使能外设时钟（RCC）                               │
│     │                                                       │
│     ├─► 4.3 初始化外设（GPIO、USART等）                        │
│     │                                                       │
│     └─► 4.4 进入 while(1) 主循环                             │
│             │                                               │
│             └─► 执行业务逻辑                                  │
│                                                             │
│  [中断发生时]                                                │
│     │                                                       │
│     ├─► 保存当前执行上下文                                    │
│     │                                                       │
│     ├─► 跳转到 stm32f10x_it.c 中的中断服务程序                  │
│     │                                                       │

│     └─► 恢复上下文，返回主程序                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 开发注意事项

### 1. 变量声明位置
在C89/C90标准中，变量声明必须放在函数开头：
```c
void func(void) {
    int a;          // ✓ 正确：声明在开头
    a = 10;
    int b;          // ✗ 错误：声明在执行语句之后
}
```

### 2. 时钟使能顺序
必须先使能外设时钟，再配置外设：
```c
// ✓ 正确
RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOC, ENABLE);
GPIO_Init(GPIOC, &GPIO_InitStruct);

// ✗ 错误：时钟未使能就配置外设
GPIO_Init(GPIOC, &GPIO_InitStruct);
RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOC, ENABLE);
```

### 3. 启动文件选择
根据芯片Flash容量选择正确的启动文件：
- C8T6 (64KB Flash) → 使用 `startup_stm32f10x_md.s`
